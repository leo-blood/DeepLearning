# 03 泛型

> 泛型让你写出"类型安全的通用代码"，是 TypeScript 最强大的特性之一。

---

## 1. 什么是泛型

不用泛型时，要么写死类型，要么退化成 `any`：

```ts
// 只能处理 string，不通用
function identity(value: string): string {
  return value;
}

// 通用了，但丢失类型信息
function identity(value: any): any {
  return value;
}
```

泛型让类型本身成为"参数"：

```ts
// T 是类型参数，调用时再确定
function identity<T>(value: T): T {
  return value;
}

const s = identity('hello');   // T 推断为 string，返回 string
const n = identity(42);        // T 推断为 number，返回 number
```

## 2. 泛型函数

```ts
function first<T>(arr: T[]): T | undefined {
  return arr[0];
}

const firstNum  = first([1, 2, 3]);    // number | undefined
const firstStr  = first(['a', 'b']);   // string | undefined
```

**项目中的例子** — `src/api/interceptor.ts` 对 Axios 响应的封装：

```ts
// Axios 本身就是泛型设计
// AxiosResponse<T> 中的 T 就是 res.data 的类型
import { AxiosResponse } from 'axios';

// 项目中的 API 函数返回 Promise<AxiosResponse<ApiResponse<T>>>
// 调用方可以精确拿到 data 的类型
const { data } = await postStudentCenterGetMyLearningTask({});
// data 的类型被推断为接口定义的具体类型，而不是 any
```

## 3. 泛型接口

**项目中最直接的例子** — Vue 的 `Ref` 和 `ComputedRef`：

```ts
// Vue 内部定义（简化版）
interface Ref<T> {
  value: T;
}

interface ComputedRef<T> {
  readonly value: T;
}
```

所以 `src/hooks/myCourse.ts` 中的接口：

```ts
interface IMyCourse {
  activeTab: Ref<LearningStatus>;        // .value 是 LearningStatus 类型
  showProgress: ComputedRef<boolean>;    // .value 是 boolean
  searchKeyword: Ref<string>;            // .value 是 string
  loading: Ref<boolean>;
  // ...
}
```

## 4. 泛型约束 `extends`

有时候想限制 T 必须有某些属性：

```ts
interface HasId {
  id: number;
}

// T 必须至少有 id 属性
function findById<T extends HasId>(list: T[], id: number): T | undefined {
  return list.find(item => item.id === id);
}

findById([{ id: 1, name: 'Alice' }], 1);  // ✅
findById(['a', 'b'], 1);                   // ❌ string 没有 id 属性
```

**项目中的例子** — 过滤函数：

```ts
// src/hooks/myCourse.ts
const businessFilter = ({ skillList }: MyLearningTaskVo): string[] =>
  skillList?.filter(Boolean).map(({ subBusinessName, schemeType }) =>
    `${subBusinessName}-${schemeType}`
  ) ?? [];
```

`filter(Boolean)` 过滤掉 falsy 值，利用了 TS 的类型守卫自动收窄 `T | undefined` → `T`。

## 5. 内置泛型工具类型

这些是 TypeScript 标准库提供的，项目中大量使用：

### `Partial<T>` — 所有属性变可选

```ts
interface IConfig {
  apiUrl: string;
  timeout: number;
}

type PartialConfig = Partial<IConfig>;
// 等价于 { apiUrl?: string; timeout?: number; }

function updateConfig(patch: Partial<IConfig>) { ... }
updateConfig({ timeout: 3000 });  // 只更新部分字段
```

### `Required<T>` — 所有可选变必填

```ts
type RequiredConfig = Required<PartialConfig>;
// 等价于 { apiUrl: string; timeout: number; }
```

### `Record<K, V>` — 键值对映射

**项目中的直接使用** — `src/hooks/myCourse.ts`：

```ts
// Record<LearningStatus, MyLearningTaskVo[]>
// 等价于 {
//   IN_PROGRESS: MyLearningTaskVo[];
//   FINISHED: MyLearningTaskVo[];
//   NOT_FINISHED: MyLearningTaskVo[];
// }
myCourseList: Record<LearningStatus, MyLearningTaskVo[]>;
```

### `Pick<T, K>` — 从 T 中挑选部分属性

```ts
interface IUser {
  id: number;
  name: string;
  email: string;
  password: string;
}

type PublicUser = Pick<IUser, 'id' | 'name' | 'email'>;
// { id: number; name: string; email: string; }
```

### `Omit<T, K>` — 从 T 中排除某些属性

```ts
type SafeUser = Omit<IUser, 'password'>;
// { id: number; name: string; email: string; }
```

### `ReturnType<T>` — 提取函数返回类型

```ts
function getConfig() {
  return { apiUrl: 'https://...', timeout: 5000 };
}

type Config = ReturnType<typeof getConfig>;
// { apiUrl: string; timeout: number; }
```

## 6. 条件类型

```ts
type IsString<T> = T extends string ? 'yes' : 'no';

type A = IsString<string>;   // 'yes'
type B = IsString<number>;   // 'no'
```

内置工具 `NonNullable<T>` 就是用条件类型实现的：

```ts
type NonNullable<T> = T extends null | undefined ? never : T;

type Clean = NonNullable<string | null | undefined>;  // string
```

**项目中用到的场景** — 可选链 + 空值合并：

```ts
// 运行时逻辑
const list = earnings.value?.subBusinessDimensionEarningsList ?? [];

// 类型层面等价于
type MaybeList = SubBusinessDimensionEarningsVO[] | undefined;
const list: SubBusinessDimensionEarningsVO[] = maybeList ?? [];
```

## 7. `Promise<T>` — 最常见的泛型

```ts
async function fetchUser(id: number): Promise<IUser> {
  const res = await axios.get(`/api/user/${id}`);
  return res.data;
}
```

**项目中的例子** — `src/hooks/myCourse.ts`：

```ts
const fetchMyCourseList = async (): Promise<void> => {
  // 不返回任何值，所以是 Promise<void>
  const { data } = await postStudentCenterGetMyLearningTask({ ... });
  myCourseList[LearningStatus.IN_PROGRESS] = data?.processingLearningTaskList ?? [];
};
```

## 练习

```ts
// 题 1：实现一个泛型函数，从对象中安全获取属性值
// 要求：key 必须是 obj 的合法属性名
function getProperty<T, K extends keyof T>(obj: T, key: K): ___ {
  return obj[key];
}

// 题 2：使用 Omit 和 Pick 定义以下类型
interface IAPIResponse {
  code: number;
  message: string;
  data: unknown;
  timestamp: number;
  traceId: string;
}

// 只保留 code 和 data
type MinimalResponse = ___;

// 去掉 traceId 和 timestamp
type SimpleResponse = ___;
```

<summary>参考答案</summary>

```ts
// 题 1
function getProperty<T, K extends keyof T>(obj: T, key: K): T[K] {
  return obj[key];
}

// 题 2
type MinimalResponse = Pick<IAPIResponse, 'code' | 'data'>;
type SimpleResponse  = Omit<IAPIResponse, 'traceId' | 'timestamp'>;
```
