# 04 函数类型与高阶函数

---

## 1. 函数的完整类型

```ts
// 参数类型 + 返回类型
function add(a: number, b: number): number {
  return a + b;
}

// 箭头函数
const multiply = (a: number, b: number): number => a * b;
```

返回类型通常可以省略（TS 自动推断），但显式写出来文档性更好，也能防止意外返回错误类型。

## 2. 可选参数与默认值

```ts
// 可选参数（用 ?）
function greet(name: string, title?: string): string {
  return title ? `${title} ${name}` : name;
}
greet('李明');         // ✅
greet('李明', '博士'); // ✅

// 默认值（自动推断类型，不需要 ?）
function createUser(name: string, role = 'user') {
  return { name, role };
}
```

**项目中的例子** — `src/hooks/performance.ts`：

```ts
// date 可选，不传时使用 salaryDate.value
const getEarnings = async (date?: Date | string) => {
```

## 3. 函数作为类型

函数可以作为类型来声明变量或参数：

```ts
// 函数类型语法：(参数类型) => 返回类型
type Handler = (event: MouseEvent) => void;
type Transformer<T, R> = (input: T) => R;

// 在接口中声明方法
interface IActions {
  onClick: (id: number) => void;
  onSubmit: (data: FormData) => Promise<boolean>;
}
```

**项目中的例子** — `src/hooks/myCourse.ts` 接口中的函数声明：

```ts
interface IMyCourse {
  goStudy: (course: MyLearningTaskVo) => void;
  fetchMyCourseList: () => Promise<void>;
  timeRangeFilter: (course: MyLearningTaskVo) => string;
  businessFilter: (course: MyLearningTaskVo) => string[];
  courseNameFilter: ({ name }: MyLearningTaskVo) => {
    type: 'default' | 'keyword';
    content: string;
  }[];
}
```

注意 `courseNameFilter` 使用了 **解构参数** 写法，入参是 `MyLearningTaskVo` 但只关心其中的 `name` 字段。

## 4. 函数重载

同一个函数名，不同参数组合返回不同类型：

```ts
function process(input: string): string;
function process(input: number): number;
function process(input: string | number): string | number {
  if (typeof input === 'string') return input.toUpperCase();
  return input * 2;
}

const a = process('hello');  // 返回 string
const b = process(5);        // 返回 number
```

## 5. `void` vs `never`

```ts
// void：函数执行完毕，不返回有意义的值（可以 return undefined）
function log(msg: string): void {
  console.log(msg);
}

// never：函数永远不会正常结束（抛异常或死循环）
function panic(msg: string): never {
  throw new Error(msg);
}
```

## 6. 高阶函数：函数接收/返回函数

**项目中的例子** — `src/hooks/myCourse.ts` 使用 `debounce`：

```ts
import { debounce } from 'lodash';

// debounce 接收函数，返回函数——典型的高阶函数
// 类型：debounce<T extends (...args: any) => any>(func: T, wait?: number): DebouncedFunc<T>
watch(searchKeyword, debounce(fetchMyCourseList, 800, { leading: true }));
```

`debounce` 的泛型 `T` 保留了原函数的参数类型，返回值也有正确的类型提示。

## 7. 回调函数模式

```ts
// 接受回调的函数
function withLoading<T>(fn: () => Promise<T>): () => Promise<T> {
  return async () => {
    try {
      loading.value = true;
      return await fn();
    } finally {
      loading.value = false;
    }
  };
}
```

这正是项目中 `try/finally` 包装 loading 的抽象形式：

```ts
// src/hooks/myCourse.ts 中反复出现的模式
const fetchMyCourseList = async (): Promise<void> => {
  if (loading.value) return;
  try {
    loading.value = true;
    const { data } = await postStudentCenterGetMyLearningTask({ ... });
    // ...
  } finally {
    loading.value = false;
  }
};
```

## 9. `async/await` 函数类型

```ts
// async 函数返回类型总是 Promise<T>
async function fetchData(): Promise<string> {
  const res = await fetch('/api/data');
  return res.text();  // text() 返回 Promise<string>，await 后是 string
}

// 即使不写 Promise，TS 也会自动推断
async function fetchCount() {
  return 42;  // 推断为 Promise<number>
}
```

```ts
// 题 1
function cache<T>(fn: () => T): () => T {
  let cached: { value: T } | undefined;
  return () => {
    if (!cached) {
      cached = { value: fn() };
    }
    return cached.value;
  };
}

// 题 2
```
