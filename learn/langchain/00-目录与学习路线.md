# LangChain 进阶学习课件

> 面向已掌握 LangChain 基础用法的开发者，深入理解核心机制、掌握生产级实践。

---

## 学习路线

```
01 记忆系统与对话链         ← 让 LLM "记住"上下文
       ↓
02 RAG 检索增强生成         ← 让 LLM "读懂"你的文档
       ↓
03 Agents 与工具调用        ← 让 LLM "动手"执行任务
       ↓
04 LCEL 与流式输出          ← 用声明式管道组合一切
       ↓
05 LangGraph 多 Agent       ← 构建有状态的 Agent 系统
```

## 推荐学习顺序

**理解核心机制**（2-3天）：`01` → `02` → `03`

**掌握工程化表达**（1-2天）：`04`（LCEL 是现代 LangChain 的基础）

**多 Agent 编排**（2-3天）：`05`

**动手实战**（贯穿全程）：三个沙箱与课件同步练习
